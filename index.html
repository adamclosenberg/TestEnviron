Claude Actions
Automated exports & reports
📤
Export to Pastel
Pulls all Confirmed orders, generates the Pastel Express v19 CSV import file, saves to Google Drive.
Say "export orders to Pastel" in Claude → CSV saved to Google Drive → import via Process › Customers › Batch › Import.
✂️
Generate Production Sheets
Aggregates order lines by style+colourway, calculates size breakdowns and fabric requirements, outputs a PDF to Google Drive.
Say "generate production sheets" in Claude → PDF saved to Google Drive.
📊
Order Summary Report
Full season summary — orders by customer, units by style, size ratios, revenue — as a PDF to Google Drive.
Say "generate order summary report" in Claude → PDF saved to Google Drive.
🔄
Refresh Dashboard
Dashboard pulls live data from Airtable on every load. Hit Refresh anytime for the latest numbers.
